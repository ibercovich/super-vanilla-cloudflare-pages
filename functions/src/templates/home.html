Super Vanilla Contact Manager

A simple, clean contact management application built with modern vanilla web technologies.

This application demonstrates how to build a fully functional web app using:

  • HTML, CSS, and JavaScript
  • Bulma CSS framework for styling
  • HTMX for dynamic interactions without complex JavaScript
  • Cloudflare Pages for hosting and serverless functions
